## Game Goals and Mechanics

### The Treasure Chest
- Matches begin with a locked chest (padlocks and magical seals).
- Players contribute real crypto into a prize pool.

### Victory Conditions
- Open Path: Remove all locks to win.
- Block Path: Seal the chest permanently to win.
- Double Agents: Choose sides, betray, or pivot to maximize rewards.

### Core Mechanics
- Forge Keys, Unlock Chests, Secure with Seals, Form and Break Alliances.
- All actions have cooldowns (2-6 minutes) to balance strategy and pace.
- Game state evolves in real-time, with live updates via Mantle smart contracts.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/12-archetypes.git
   cd 12-archetypes
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure environment variables in `.env.local`:
   ```bash
   NEXT_PUBLIC_THIRDWEB_CLIENT_ID=your_client_id_here
   ```
